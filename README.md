# Image Edge Detection Project

This project demonstrates basic image processing techniques, including noise addition, smoothing, and edge detection.

## About
The program starts with a grayscale image, adds Gaussian noise, and then applies smoothing using a Gaussian filter to reduce noise.

After that, multiple edge detection techniques are applied:
- Sobel filter
- Prewitt filter
- Laplacian
- Laplacian of Gaussian (LoG)

The results are then converted into binary images using thresholding and zero-crossing methods.

## Purpose
The goal of this project is to compare different edge detection methods and understand how they behave on noisy images.

## Tools Used
- Python
- NumPy
- Matplotlib
- SciPy
- scikit-image
